# TP1_bigdata
1. وصف المشروع بصفة عامة
هذا المشروع هو تطبيق لاستخراج بيانات الويب (Web Scraping) باستخدام لغة بايثون، يهدف إلى جمع معلومات منظمة عن إنترنت الأشياء (Internet of Things - IoT) من مصادر موثوقة على الويب.

يقوم البرنامج بزيارة صفحات ويب متخصصة في مجال IoT، واستخراج المحتوى النصي (عناوين، فقرات، عناصر قوائم) وتنظيفه، ثم تصنيفه حسب الفئة التقنية، وأخيرًا حفظ النتائج في ملف CSV منظم 
يصلح للتحليل أو الاستخدام في مشاريع تعلم الآلة أو قواعد البيانات.

2. نوع الموقع المستخدم (Website Type)
الموقع المستخدم هو: postscapes.com

وصف الموقع:
موقع تقني متخصص في إنترنت الأشياء (IoT) والتقنيات المتصلة.
يقدم أدلة شاملة (Guides) ومقالات عن:
بروتوكولات الاتصال (Communication Protocols)
الأجهزة والمكونات المادية (Hardware)
منصات IoT السحابية (Cloud Platforms)
أمان وحماية أجهزة IoT (Security)
المعايير القياسية (Standards)
